# StudyFlow — AI-Powered Learning Platform

StudyFlow is an all-in-one learning platform designed to help students actively understand and retain concepts. It converts static study material into interactive quizzes, provides performance insights, and supports focused study sessions.

---

## Features

### AI Quiz Engine
- Upload PDF notes or textbooks
- Automatically generate multiple-choice questions using AI
- Instant feedback with explanations
- Bookmark difficult questions for revision

### Performance Analytics
- Topic-wise performance visualization (spider graph)
- Accuracy tracking over time
- AI-based mastery insights
- Study activity heatmap

### Focus and Productivity
- Pomodoro timer for structured study sessions
- Daily goal tracking
- Focus time monitoring
- Exam countdown feature

### StudyBot AI
- AI assistant for doubt solving
- On-demand concept explanations
